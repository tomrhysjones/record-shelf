# 📖 Overview

Record Shelf is a full-stack web application built with Django that allows users to build and manage a personal music collection.

Users can create an account, catalogue albums, rate records, write personal reviews and browse collections through a clean and responsive interface.

The project demonstrates user authentication, relational database design, full CRUD functionality and server-side rendering using Django.

---

# ✨ Features

- User authentication
- Create, read, update and delete albums
- Personal record collection
- Album ratings
- Notes and reviews
- Genre filtering
- Search functionality
- Responsive design
- Protected user content

# 🔐 Authentication

Users can register for an account, securely log in and manage their own personal collection.

Authenticated users can create, edit and delete only the albums that belong to their own library.

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/tomrhysjones/record-shelf.git
```

Navigate into the project:

```bash
cd record-shelf
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Apply migrations:

```bash
python manage.py migrate
```

Run the development server:

```bash
python manage.py runserver
```
